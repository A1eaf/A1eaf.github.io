Arch Linux安装rime输入法并使用雾凇拼音词库

发表于 2025-07-12 分类于 Linux系统

“雾凇拼音”是近年比较热门的开源词库，整合了多个优秀词库并持续更新，并且雾凇拼音支持动态词频调整，常用词会越输越准，所以特别适合作为Linux系统下中文用户词库的最佳选择！不过Linux下配置Rime比较麻烦，所以就有了这边傻瓜版的Fcitx5框架下rime输入法的安装指北。

一、安装Fcitx5并配置rime

1.安装Fcitx5框架，pacman 会提醒你需要安装 fcitx5、fcitx5-configtool、fcitx5-gtk、fcitx5-qt。这里直接直接默认全部安装即可

sudo pacman -S fcitx5-im

2.在Fcitx5框架下使用rime输入法

sudo pacman -S fcitx5-rime

3.修改环境配置处理全局输入

sudo nano /etc/environment

按照下面修改变量值，不存在就添加环境变量

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
SDL_IM_MODULE=fcitx
GLFW_IM_MODULE=ibus

二、通过paru安装雾凇拼音词库

雾凇拼音（rime-ice）整合了多个优秀词库，支持现代词汇、成语、古诗词等。如果有其他问题，建议直接去GitHub上看大佬的配置说明，写的很全。

1.这里使用paru直接安装

paru -S rime-ice-git

如果想只用雾凇词库的拼音输入法，可以执行下面的命令

paru -S rime-ice-pinyin-git

2.通过编辑用户自定义配置文件启用雾凇拼音

nano ~/.local/share/fcitx5/rime/default.custom.yaml

新增或修改以下配置

patch:
    schema_list:
        - schema: rime_ice
    menu/page_size: 9             # 每页候选词数量

启用雾凇拼音输入方案

nano ~/.local/share/fcitx5/rime/rime_ice.custom.yaml

新增或修改以下配置

patch:
    __include: rime_ice:/patch  # 关键！加载雾凇拼音的增强配置

    # 可选自定义项
    switches:  
        - name: ascii_mode
        reset: 0                # 默认中文输入
        states: [ 中文, 西文 ]
    translator/enable_user_dict: false  # 禁用个人词库（推荐用雾凇词库）

三、部署与激活

任务栏
    重新部署配置：
        Fcitx5：右键任务栏图标 → 中州韵 → 雾凇拼音 → 重新部署

挂载rime的同步文件夹
1 首先安装samba服务：

sudo pacman -S samba

2 创建凭证文件：

sudo mkdir /etc/samba/credentials #创建凭证目录
sudo nano /etc/samba/credentials/syncrime

编辑证书文件加入访问smb共享需要的用户名密码如下格式：

username=#当前用户名称
password=#当前用户密码

5 创建挂载脚本，并设置开机自启动。要注意挂载后的目录权限，且创建脚本后不可移动，否则上下文关系变更后脚本无法开机执行。

#!/bin/bash
sudo -S sudo mount -t cifs -o credentials=/etc/samba/credentials/syncrime,iocharset=utf8,uid=aleafyu,gid=wheel 远程目录地址 ~/.local/share/fcitx5/rime/sync/  << EOF

#执行提权命令时， 在此设置当前用户的的密码

EOF
